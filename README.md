## Project Description
This project involved the design and creation of a **travel website** advertising a selected country. The website was designed using **Figma** and supported by **Photoshop image manipulation**. The focus of the project was effective visual communication, layout design, and consistent use of a self-created style guide.

---

## Website Requirements
- Five pages:
  - Home / About  
  - Eat  
  - Where to Stay  
  - Festivals & Culture  
  - Attractions
- Five working navigation buttons
- Consistent layout, typography, and colour scheme
- Clear text and relevant imagery on each page
- Photoshop-edited images displayed throughout the site

---

## Photoshop Image Manipulation
The project demonstrates **five different Photoshop effects**, including:
- Image compositing  
- Visual illusion  
- Glow effect  
- Double exposure  
- Cartoon or dispersion effect  

Each effect was selected to enhance the visual impact of the website and support the travel theme.
